## Situation

Adler Werbe-Geschenke in Bregenz ist ein etablierter B2B-Anbieter für:
- Werbeartikel (Kugelschreiber, Tassen, Taschen)
- Textilveredelung (Aufdruck, Stickerei, Transferdruck)
- Giveaways für Messen und Events
- Mitarbeiter- und Kundenpflege-Geschenke
- Hotellerie-Service

**Problem:** Adler hat **KEINE Website**. Das ist ein kritisches Problem für einen B2B-Anbieter, weil:

1. **Google-Suchende finden sie nicht** — Unternehmen googlen potenzielle Lieferanten vor dem Erstkontakt
2. **Keine Produktvisuelle** — Der erste Eindruck ist eine leere Suchergebnisseite
3. **Wettbewerber gewinnen** — Wer eine professionelle Seite hat, bekommt die Anfrage
4. **Ausschreibungen** — Viele KMUs kaufen nur bei Lieferanten, die sie online „verifizieren" können

## Was diese Website löst

Eine **professionelle B2B-Katalog-Website** mit:

- ✅ Hero mit allen Leistungen auf einen Blick
- ✅ Leistungsübersicht (6 Kategorien)
- ✅ Produkt-Highlights mit Preisen
- ✅ Branchenlösungen (Hotels, Restaurants, Unternehmen…)
- ✅ 3 Vorteile (Erfahrung seit 1950, Top-Qualität, Rasche Lieferung)
- ✅ 4-Schritt-Prozess (Beratung → Angebot → Produktion → Lieferung)
- ✅ Kontaktformular + Google Maps
- ✅ Professionelles Design: Navy + Gold, Montserrat + DM Sans
